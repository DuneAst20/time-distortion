# ⏳ Time Distortion

> *"I didn't scroll a website... I traveled through time."*

**Time Distortion** is an experimental, cinematic web experience that breaks away from traditional scroll-based navigation. Instead of moving down a page, every interaction manipulates the flow of time within a 3D particle universe. Built entirely without external frameworks, this project relies on raw math and vanilla web technologies to create a high-performance, emotional digital journey.

-----

## 🌌 The Experience

Navigate a timeline that evolves visually, mechanically, and emotionally depending on where—and when—you are:

  * **⏪ The Past:** Scroll up to enter a desaturated, glitchy realm. Colors fade, the text distorts with CSS keyframes, and the environment moves in reverse.
  * **⏸️ The Present:** Return to the center for a moment of clarity. The interface stabilizes, and time moves at a natural, resting pace.
  * **⏩ The Future:** Scroll down to accelerate into a high-saturation, neon-drenched horizon where particles pulse with life and the unwritten future forms.

## 🕹️ Controls & Interactions

The interface responds directly to your physical energy:

  * **Scroll Wheel:** Controls the direction of time. Scrolling up pulls you backward; scrolling down pushes you forward.
  * **Cursor Velocity:** Acts as a temporal catalyst. Fast mouse movements speed up the flow of existence, creating kinetic motion blur.
  * **Stillness:** Stopping all movement causes the timeline to freeze into a "zero-point" suspended state.

## 🛠️ Tech Stack

This project was built with a strict "no-framework" philosophy to ensure maximum performance and lightweight deployment:

  * **HTML5 `<canvas>`:** Renders the custom 3D particle projection engine.
  * **Vanilla JavaScript:** Handles scroll interception, velocity tracking, and the mathematical Z-axis physics loop.
  * **CSS3:** Powers the cinematic post-processing (hue shifts, grayscale, sepia, contrast) and text glitch effects without the overhead of WebGL shaders.

## 🚀 How to Run Locally

Because this project uses pure web standards, there are no dependencies to install or build steps to run.

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/time-distortion.git
    ```
2.  Navigate to the project folder.
3.  Open `index.html` directly in any modern web browser.

*(Alternatively, you can just download the ZIP file, extract it, and double-click `index.html`\!)*
